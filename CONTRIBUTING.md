# Contributing to AstraForge: The Sovereign AI IDE

Welcome to the development of AstraForge. Our philosophy is built on Sovereign AI - creating tools that empower developers with autonomous, privacy-focused, and capable AI assistance.

## Getting Started

This project is built with TypeScript.

### Prerequisites
- Node.js (v18+)
- npm

### Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development mode:
   ```bash
   npm run dev
   ```

## Project Structure
- `src/core`: Core logic and orchestration
- `src/agents`: AI Agent definitions

We welcome pull requests that align with our mission of AI sovereignty.
